## Requirements

- Node 0.12.0
- Express 4.13.3
- Angular 1.4.7
- Mongoose 4.2.5
- Socket.io 1.3.7
- Grunt-Docco 0.4.0
- Body-Parser 1.14.1


## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```

### Packaging Documentation

From within the root directory:

```sh
grunt docco
```
